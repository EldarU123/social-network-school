ISP Connect



What is this for?

This website is a chat style website for the ISP community to communicate during this lockdown time. It's aimed at teachers to communicate with ISP parents and for parents to communicate with each other about events happening relating to the school. If you are a parent, go to the Parents to Parents page and chat with other parents live! If you are teacher go to the teachers to parents page and chat to and with parents. The layout of this page is very loose as it is a chat style website. Have Fun!
